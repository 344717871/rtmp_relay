# rtmp_relay
rtmp relay server：
==================================================
* Platform：linux

* How to use?
example:
xxx# ./rtmp_relay rtmp://x.x.x.x/live/source rtmp://x.x.x.x/live/target
first parameter: source rtmp server url
second parameter: destination rtmp server url
